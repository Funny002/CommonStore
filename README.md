# Common Store

基于 Immutable.js 实现的一个状态管理库，通过架构分离 Data、Action、Plugin 三个模块实现完整的状态管理器。

Store 为模块化的方法封装，通过 Store 提供的 use 方法注入插件到插件模块，插件模块将解析内容向 Action 注入方法，当用户通过 Action 操作 Data 时，触发相对应的方法。

> `Data` 用于数据存储。
>
> `Action` 用于数据处理，通过 Action 可以对 Data 进行操作。
>
> `Plugin` 用于插件的扩展，比如日志、异常处理、数据持久化等等。

## 特性

- 🔒 **不可变数据**: 基于 Immutable.js，确保状态的可预测性和一致性
- 🧩 **插件化架构**: 灵活的插件系统，支持功能扩展和自定义
- 📦 **模块化设计**: Data、Action、Plugin 三者职责分离，代码清晰易维护
- 🎯 **类型安全**: 完整的 TypeScript 类型支持，开发体验更佳
- 🚀 **现代语法**: 使用 ES2023+ 和 TypeScript 5.x，享受最新语言特性

## 架构

```
┌──────────────────────────────────────────────────┐
|                      Store                       |
├──────────────────────────────────────────────────┤
|  ┌────────────┐  ┌────────────┐  ┌────────────┐  |
|  |    Data    |  |   Action   |  |   Plugin   |  |
|  └────────────┘  └────────────┘  └────────────┘  |
└──────────────────────────────────────────────────┘
```
